# LLM SEO Agent

A comprehensive SEO tool that uses LLMs to track keyword rankings and perform competitive analysis.

## Features

- Track keyword rankings across multiple LLMs
- Schedule automated audits
- Competitive analysis with AI-powered insights
- Compare your company against competitors
- Daily automated reports

## Setup

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`

## Usage

1. Add your LLM API keys in Settings
2. Create keyword groups and add keywords to track
3. Set up competitive analysis with competitors
4. Schedule automated runs for daily monitoring
5. View results in the dashboard

## Components

- **Keywords & Groups**: Manage your tracking keywords
- **Active Runs**: Real-time audit dashboard
- **Scheduled Runs**: Automated audit scheduling
- **Competitive Analysis**: Competitor comparison tools
- **Reports**: Historical data and insights
- **Settings**: API key management
